# Webcam Overlay

Windows 雙攝影機浮動顯示工具（第一階段）。

## 功能

- 最多指定兩個攝影機來源
- 隱藏、顯示一個、顯示兩個三種狀態
- 滑鼠拖曳浮動視窗，滾輪等比例縮放
- 單畫面點擊切換來源；雙畫面點擊交換來源
- 全域快速鍵：`Ctrl+Alt+C`、`Ctrl+Alt+0/1/2`、`Ctrl+Alt+S`
- 浮動視窗置頂與位置保存

## 開發執行

```powershell
npm.cmd install
npm.cmd start
```

## 建置免安裝版

```powershell
npm.cmd run build:portable
```

成品位於 `release` 資料夾。

## 第二、第三階段

- 矩形、圓角矩形與圓形外觀
- 框線顏色及粗細
- 攝影機背景：保留原背景、背景模糊、人物去背
- 水平鏡像、完整顯示或裁切填滿
- 系統匣快速控制
- 攝影機中斷提示與每 5 秒自動重連
- 多螢幕拖曳、固定比例與平滑滾輪縮放
